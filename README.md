# Tugas Besar 1 IF2010 Pemrograman Berorientasi Objek

Repository ini berisi struktur dasar proyek C++ untuk tugas besar.

## Struktur

- `include/`: header file
- `src/`: source file
- `config/`: file konfigurasi
- `save/`: file penyimpanan runtime

## Build

```bash
make
```

## Run

```bash
./bin/nimonspoli
```
